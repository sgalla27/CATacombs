# Project Description

This project is a game built in Unity programming in C# worked on by Sean Gallagher, Katie Harrington, Lily Sweeney, and Arielle Korn. The game follows the protagonist Will D. Cat, 
the mascot of Villanova University trying to escape the depths of the school.

# Role in Project

In this project, I worked on the programming for the UI elements and audio. In addition, I adjusted the scripts for the collisions and player movement.
